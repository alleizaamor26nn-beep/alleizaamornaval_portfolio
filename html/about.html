Behind the code & creativity

I'm a recent graduate with a burning passion for building efficient and impactful systems. Throughout my academic journey and internships, I have successfully developed several key projects that solve real-world problems — from cooperative management to educational platforms.

My featured projects include a Credit Cooperative System built with Laravel and Tailwind CSS to handle member remittances and loan processing, and a Violation Record Management System for Trimex Colleges using C# that features automated email notifications and data analytics. I also co-developed an All-in-One Point of Sale System using Flutter and PHP for Food & Beverage and Bread & Pastry departments, and designed a fully responsive Learning Management System (LMS) front-end. Additionally, I have built various inventory systems and POS applications using VB.NET, HTML/CSS, and MySQL.

I have solid experience working with databases using MySQL through phpMyAdmin and XAMPP, and I'm knowledgeable in Java using Apache NetBeans. I enjoy designing clean, user-friendly interfaces and consider myself more inclined toward front-end development, while still understanding the logic behind system and database design. I am eager to apply my skills, continue learning, and contribute to meaningful software solutions in a professional environment.
